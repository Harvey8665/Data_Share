## STM32模板

### STM32工程模板的结构如下：<br>
    （某型号）
        Core：STM32库函数驱动
        FWLib：STM32库函数源文件
        Hardware：放置工程中需要用到的模块的源文件
        OBJ：放置编译产生的文件
        SYSTEM：包含正点原子的常用的三个文件
        User：工程文件所在文件夹

工程模板采用`7Zip`打包，请使用`7Zip`或其他可以解压`7z`后缀压缩包的工具进行解压<br>
`7Zip`开源软件下载地址：https://www.7-zip.org/<br>
